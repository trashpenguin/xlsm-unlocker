Frequently Asked Questions

Does this work on e-IPCRF files?

Yes! This tool was specifically built to unlock e-IPCRF .xlsm files used by DepEd teachers in the Philippines. It removes the finalization locks so you can continue editing.

Will the macros still work after unlocking?

Yes, the VBA macros inside the file are untouched. Only the XML-level protection tags are removed. Note: avoid clicking the "Finalize" button inside the form or it will re-lock the file.

Is my data safe?

Absolutely. This tool uses JSZip to process your file entirely in your browser. Nothing is sent to any server. You can even use it offline after the page has loaded.

What if the file still can't be edited after unlocking?

Some cells may be individually marked as "locked" in their cell format. In Excel, go to Review → Unprotect Sheet, or select all cells → Format Cells → Protection → uncheck "Locked".

Can I unlock multiple files at once?

Yes! Drop or select multiple .xlsm/.xlsx files at the same time. All files are processed in your browser and you can download them individually or as a single ZIP archive.
